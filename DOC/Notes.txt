Modal - contains the class No and Pilha, represent our object. Make all Pilha's operations, including : Add and Remove.

Controller - contains the class Text. Work with the text and information that the user insert, creating others funtions to return to our View. Receive data from View and use the function of Modal to manupulate our Pilha.
